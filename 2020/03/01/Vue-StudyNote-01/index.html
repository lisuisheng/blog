Vue学习要点回顾__01

  1. MVC 和 MVVM 的区别

  2. 插值表达式

    v-cloak   v-text   v-html   v-bind(:)  v-on(@)  v-model   v-for   v-if  v-show。

  3. 事件修饰符 

    .stop .prevent .capture .self .once。

  4. el 指定要控制的区域，data 是个对象，指定了控制的区域内要用到的数据，methods 虽然带个s后缀，但是是个对象，这里可以自定义了方法。

  5. 在 VM 实例中，如果要访问 data 上的数据，或者要访问 methods 中的方法，必须带 this。
  6. 在 v-for 要会使用 key 属性 （只接受 string / number）。
  7. v-model 只能应用于表单元素。
  8. 在vue中绑定样式两种方式 v-bind:class 、v-bind:style。
